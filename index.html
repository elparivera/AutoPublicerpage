¿Dónde quieres publicar tu sitio? 🌐

Las 4 plataformas son 100% gratuitas y el proceso es completamente automático.

🐙 GitHub Pages: Crea repositorio, sube archivos, configura CI/CD, activa Pages, genera SEO, LICENSE, README y verifica que el sitio esté en vivo — completamente automático.
⚡ Netlify: Sube archivos directamente a la CDN global. Sitio en vivo en segundos con URL tu-sitio.netlify.app — sin necesidad de Git.
▲ Vercel: Despliega en la Edge Network global de Vercel. URL instantánea tu-sitio.vercel.app. Deploy atómico con rollback automático.
☁️ Cloudflare Pages: Publica en más de 300 datacenters globales de Cloudflare. URL tu-sitio.pages.dev. Velocidad máxima garantizada.
🔥 Firebase Hosting: Despliega directamente en la infraestructura de Google. URL tu-sitio.web.app con SSL automático y CDN global. Tier gratuito generoso.
🤗 Hugging Face Spaces: Crea un Space estático con tu HTML puro. URL usuario.hf.space/repo — gratis, con CDN global y sin necesidad de Git local.

Conecta tu cuenta GitHub

Necesitas un Token de Acceso Personal (PAT) para operar automáticamente.

El link de abajo abre GitHub con los permisos ya pre-seleccionados:

  1. Clic → Generar token con permisos listos ↗
  2. Desplázate hasta abajo y haz clic en "Generate token"
  3. Copia el token (empieza con ghp_) y pégalo aquí

Conecta tu cuenta Netlify ⚡

Solo necesitas un Personal Access Token de Netlify. Es gratuito.

Sigue estos pasos para obtener tu token:

  1. Clic → Ir a Netlify → Tokens ↗
  2. Clic en "New access token" y ponle un nombre (ej: AutoPublisher)
  3. Copia el token y pégalo aquí abajo
🔒 Tu token se usa directamente en el navegador para llamar la API de Netlify. Nunca se envía a ningún servidor externo.

Conecta tu cuenta Vercel ▲

Solo necesitas un Access Token de Vercel. La cuenta es gratuita.

Pasos para obtener tu token:

  1. Clic → Vercel → Account → Tokens ↗
  2. Clic en "Create", escríbele un nombre (ej: AutoPublisher)
  3. Copia el token generado y pégalo aquí abajo
🔒 Tu token se usa directamente en tu navegador. Nunca se envía a servidores externos.

Conecta tu cuenta Cloudflare ☁️

Necesitas tu Account ID y un API Token con permisos de Pages.

Pasos para obtener tus credenciales:

  1. → API Tokens → Create token ↗ — usa la plantilla "Edit Cloudflare Pages"
  2. Tu Account ID: dash.cloudflare.com → barra lateral derecha bajo tu dominio
🔒 Credenciales usadas solo en tu navegador local. Nunca se transmiten a terceros.

Conecta tu proyecto Firebase 🔥

Necesitas un token CI y el Project ID de Firebase. Ambos gratuitos.

Pasos para obtener tus credenciales:

  1. → Firebase Console → Crear proyecto ↗ (o usa uno existente)
  2. Instala Firebase CLI: npm install -g firebase-tools
  3. Ejecuta firebase login:ci — copia el token generado
  4. Tu Project ID lo ves en la URL de la consola: console.firebase.google.com/project/tu-project-id
🔒 El token se usa directamente en tu navegador para llamar la REST API de Firebase Hosting. Nunca se envía a servidores de terceros.

Conecta tu cuenta Hugging Face 🤗

Necesitas un Access Token de Hugging Face con permisos de escritura. Es 100% gratuito.

Pasos para obtener tu token:

  1. Clic → Hugging Face → Nuevo token de escritura ↗
  2. Ponle un nombre (ej: AutoPublisher) y selecciona Write
  3. Copia el token (empieza con hf_) y pégalo aquí
🔒 Tu token se usa directamente en el navegador para llamar la API de Hugging Face Hub. Nunca se envía a servidores externos.
